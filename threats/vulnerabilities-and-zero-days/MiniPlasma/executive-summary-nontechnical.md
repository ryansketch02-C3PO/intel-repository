# MiniPlasma (CVE-2020-17103) — Executive Summary (Non-Technical)

A Windows security vulnerability that Microsoft said it fixed back in 2020 turns out to still be exploitable today. A researcher named Nightmare-Eclipse — who has been publicly releasing Windows exploits in recent weeks — published a working attack tool that lets any standard Windows user gain full administrator (SYSTEM) control of the machine. It has been confirmed to work against fully updated Windows 11 computers as of May 2026. Microsoft has not yet released a fix or made a public statement.
